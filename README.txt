HOW TO RUN
----------
1. Via .exe file
   - Go to main folder in root directory
   - Search for file named main.exe with cat icon
   - Double click and it will run automatically in console
2. Via .py file
   - Open the main.py file in the root directory
   - Run main.py file in the console with command:
     python main.py
     then enter
   - It will open and run automatically in console
